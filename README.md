# PrepTrack

PrepTrack is a preparation tracking application designed to help learners organize, monitor, and optimize their study progress. It provides a structured way to track preparation activities, manage learning resources, and stay consistent while working toward academic or professional goals.

🔗 **Live Demo:** https://preptrack-rho.vercel.app/

---

## Overview

PrepTrack is built to support students and professionals who want a simple yet effective system to track their preparation journey. Whether you are preparing for technical interviews, academic exams, or professional certifications, PrepTrack helps you maintain visibility over your progress and stay focused on your goals.

The application emphasizes clarity, simplicity, and usability so users can concentrate on learning rather than managing complex tools.

---

## Features

- Authentication – User signup/login with profile management.

- Create Sheets – Users can create custom DSA sheets for structured preparation.

- Add Questions – Add and organize coding questions by topic, difficulty, or company.

- Mark Progress – Mark questions as Solved, Attempted, or Revisit and track completion.
